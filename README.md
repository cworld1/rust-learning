## Local Development

Environment requirements:

- [Rust](https://www.rust-lang.org/): 1.78+

Clone the repository:

```shell
git clone https://github.com/cworld1/rust-learning.git
cd ./rust-learning
```

Run the code:

```shell
cd <code_folder>
cargo run
```

> For 1.2 part, please use `rustc main.rs && main` instead to run the code.
